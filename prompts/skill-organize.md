# Skill Organize

Reorganize skills hierarchy for optimal retrieval.

## Instructions

1. Read `{vault_path}/skills/_tree.md` and `{vault_path}/skills/_schema.md`.
2. Read all skill files in `{vault_path}/skills/`.
3. Merge similar skills, split multi-topic ones, reclassify misplaced ones. Target 3-7 items per folder.
4. Update `_tree.md` and `_stats.md` (counts, merges/splits performed, date).

Preserve skill content — only change structure and metadata. Prefer clear names over clever ones.
